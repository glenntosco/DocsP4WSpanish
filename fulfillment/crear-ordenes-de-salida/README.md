# Creación de Ticket de Picking

Manual de creación de ticket de picking

{% hint style="warning" %}
Un Pick Ticket es un pedido de venta sin precio, como P4 Warehouse es un sistema de gestión de almacenes no tenemos información de precios. La información de precios se gestiona en su ERP.
{% endhint %}

En Cumplimiento seleccione Recoger tickets.&#x20;

Cuando se abra el cuadro de selección de tickets, seleccione «Nuevo ticket de selección» en el cuadro desplegable de entrada de datos.

![](<../../.gitbook/assets/image (199).png>)

{% hint style="info" %}
A medida que el Ticket de selección avanza por cada paso, los botones se hacen visibles u ocultos para evitar confusiones.
{% endhint %}



Seleccione el cliente, el almacén y deje en blanco el número de ticket de selección si desea que el sistema seleccione automáticamente el siguiente de la serie.&#x20;

A continuación, Envíe y se creará el borrador del ticket de selección.

![](<../../.gitbook/assets/image (126).png>)

En la pantalla de selección de tickets verá un nuevo ticket de selección. Haga clic en el número de ticket Pick para revisar/editar/añadir notas.

![](<../../.gitbook/assets/image (168).png>)

Añadir elementos al ticket Pick:

![](<../../.gitbook/assets/image (200).png>)

![](<../../.gitbook/assets/image (96).png>)

Una vez completada la entrada del ticket Pick, el usuario puede «Liberar al piso» o «Asignar» solamente.
