# Onda

La onda manual se realiza a través del menú Pick ticket y puede hacerse como una sola orden o agrupada utilizando la pestaña Pick tickets.

Recoger ticket agrupado por onda:

![](<../.gitbook/assets/image (239).png>)

{% hint style="info" %}
Se puede activar o desactivar el procesamiento automático de las distintas fases de cumplimiento en Configuración->Sistema->Configuración->Empresa->Cumplimiento->Automatización.
{% endhint %}

Recoger ticket por Ola única:

![](<../.gitbook/assets/image (106).png>)

![](<../.gitbook/assets/image (105).png>)



Después de seleccionar Wave, aparecerá una ventana emergente con los valores predeterminados del sistema para Wave que se pueden cambiar si es necesario. A continuación, seleccione Enviar.

![](<../.gitbook/assets/image (70).png>)

El ticket de picking está ahora en estado Ondulado y la sección Totes es ahora visible:

![](<../.gitbook/assets/image (181).png>)

![](<../.gitbook/assets/image (204).png>)

{% hint style="info" %}
Explicación de la etiqueta Tote: Una etiqueta de contenedor puede ser una etiqueta de caja o una etiqueta de pallet, dependiendo de cómo se realice el envío. Un ejemplo sería una empresa que envía pedidos de comercio electrónico, cada caja va a un cliente diferente, en este caso la etiqueta de contenedor se aplica a cada caja.

Nota: si necesita etiquetas para cajas, seleccione Cartón al ondular. Si necesita etiquetas para pallets, seleccione Etiquetas para pallets en el momento de la impresión.
{% endhint %}
