# Envío

Esta página de configuración incluye ID de Conductor Predeterminado (se utilizará al realizar el envío; el valor introducido aquí se utilizará durante el envío automático) e ID de Vehículo Predeterminado (se utilizará por defecto al realizar el envío; el valor introducido aquí se utilizará durante el envío automático).

![](<../.gitbook/assets/image (198).png>)



{% hint style="info" %}
Se puede activar o desactivar el procesamiento automático de las distintas fases de cumplimiento en Configuración->Sistema->Configuración->Empresa->Cumplimiento->Automatización.
{% endhint %}
