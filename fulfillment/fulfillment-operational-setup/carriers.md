# Transportistas

Configuración del cumplimiento > Transportistas

Al entrar en la configuración de cumplimiento para los transportistas verá una lista como en la foto de abajo.

![P4 Warehouse Lista de transportistas](<../../.gitbook/assets/image (72).png>)

Haga clic en el botón Nuevo para añadir un nuevo transportista.

![P4 Warehouse Nueva configuración del transportista](<../../.gitbook/assets/image (57).png>)

Haga clic en Enviar para guardar la configuración.

{% hint style="info" %}
Por favor, asegúrese de introducir el código SCAC correcto en la pantalla de configuración del transportista, este dato es crítico para el funcionamiento del Conocimiento de Embarque.
{% endhint %}
