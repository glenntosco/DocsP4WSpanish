# Cliente

Configuración de Clientes, aquí es donde configurará la información de sus clientes. Por favor, complete tantos detalles como sea posible, ya que estos datos se utilizan en diversas áreas del Sistema de Almacén P4.

{% hint style="info" %}
Si NO está utilizando multiempresa o 3PL no habrá nombres de clientes en la lista. Esto es por diseño...
{% endhint %}

![P4 Warehouse Lista de clientes](<../../.gitbook/assets/image (134).png>)

Haga clic en el botón Nuevo para añadir un nuevo cliente o haga clic en el texto azul de la columna Número de cliente para editar un cliente existente.
