# Perfiles de cartonización

Entre en la Configuración del Perfil de cartonización, aquí creará diferentes perfiles dependiendo de la normativa de Cumplimiento de su cliente, o transportista de pequeña paquetería.&#x20;

Ejemplo: FedEx tiene un límite de peso de 150lb por caja.

![P4 Warehouse Configuración del perfil de encartonado](<../../.gitbook/assets/image (65).png>)

Pulse el botón Nuevo para crear un nuevo perfil.

![P4 Warehouse Configuración del perfil de encartonado](<../../.gitbook/assets/image (120).png>)

En esta pantalla asigne un nombre que le resulte comprensible y, a continuación, complete los ajustes para crear el perfil según sus especificaciones. A continuación, haga clic en Enviar para guardar los datos.

{% hint style="success" %}
Recuerde que en la pantalla de configuración del cliente deberá asignar el perfil de encartonado correcto al cliente.
{% endhint %}
