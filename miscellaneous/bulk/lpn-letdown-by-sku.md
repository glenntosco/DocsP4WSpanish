# Bajada de LPN por Sku

Al igual que el proceso de Bajada de LPN por ubicación, el proceso de Bajada de LPN por SKU mueve los LPN a ubicaciones seleccionables. Sin embargo, en la Bajada de LPN por Sku el sistema les dirigirá a todas las ubicaciones necesarias para bajar una SKU específica.

![](<../../.gitbook/assets/image (148).png>)

Para iniciar el proceso, escanee la etiqueta de baja y siga las instrucciones que aparecen en la parte inferior del dispositivo para procesar la solicitud de bajada.
