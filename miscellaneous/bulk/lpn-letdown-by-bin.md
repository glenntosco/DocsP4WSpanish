# Bajada de LPN por Bin

Bajada de LPN por Bin es el proceso en el que el usuario mueve un LPN de una ubicación de exceso de existencias a una ubicación de recogida.

![](<../../.gitbook/assets/image (241).png>)

Para iniciar el proceso, escanee la etiqueta de bajada y siga las instrucciones que aparecen en la parte inferior del dispositivo para procesar la solicitud de bajar.
