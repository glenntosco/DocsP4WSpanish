# Convertidor de tamaño de paquete

El convertidor de tamaño de paquete es un proceso de división de paquetes de mayor tamaño para pedidos específicos.



El usuario recibe una etiqueta de desglose del tamaño del envase, escanee la etiqueta para iniciar el proceso.

![](<../../.gitbook/assets/image (250).png>)

El usuario será enviado a un contenedor para retirar una o varias cajas y trasladarlas a la zona de una unidad para su recogida unitaria.
