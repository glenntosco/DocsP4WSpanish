# Recuento de ciclos por producto

El Conteo de Ciclo por Producto es un proceso donde el sistema lo dirigirá a cada contenedor que tenga un producto específico. Al entrar en el menú de recuento cíclico por producto de su PDT, se le pedirá que escanee un producto y, a continuación, el sistema de almacén P4 empezará a dirigirle a las ubicaciones específicas que tendrá que contar.

{% hint style="warning" %}
Cuando se utiliza el proceso de recuento cíclico por producto, recuerde que es fundamental contar TODAS las ubicaciones con el producto, o perderá el producto que no haya contado. Si se le dirige a un contenedor de SKU mixto, cuente únicamente el SKU seleccionado originalmente.
{% endhint %}
