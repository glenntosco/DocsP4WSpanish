# Movimiento de producto

El movimiento de productos es el proceso en el que un trabajador mueve productos de un contenedor a otro.

{% hint style="info" %}
Hay varias razones para mover el producto por el almacén, sin embargo la mejor práctica es recibir el producto y no moverlo hasta el momento de recoger un pedido de venta, hay menos posibilidades de error y menos posibilidades de que el producto se dañe.
{% endhint %}

Seleccione la opción Desplazar producto y, a continuación, escanee la primera ubicación en la que está asignado actualmente el producto.

![](<../../.gitbook/assets/image (104).png>)

Ahora escanee el producto

![](<../../.gitbook/assets/image (209).png>)

Introduzca ahora la cantidad que va a desplazar

![](<../../.gitbook/assets/image (262).png>)

Ahora escanea el contenedor donde vas a mover el producto

![](<../../.gitbook/assets/image (103).png>)

El Almacén P4 confirmará que el producto ha sido trasladado

![](<../../.gitbook/assets/image (229).png>)

{% hint style="info" %}
Dependiendo de su configuración específica, se le permitirá o no mover productos asignados a un pedido de cliente.
{% endhint %}
