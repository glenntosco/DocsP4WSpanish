# Foto del producto

Con el PDT, el usuario puede subir fotos de los productos al Almacén P4 utilizando la cámara del PDT. el primer paso es escanear el código de barras del producto y, a continuación, se abrirá la aplicación de la cámara para permitirle capturar la foto y subirla al servidor.

{% hint style="warning" %}
Configura la cámara en calidad media o en el rango de 5mp. Hay un límite de 5mb por foto.
{% endhint %}

![P4 Warehouse Product Photos](<../../.gitbook/assets/image (114).png>)

El paso 2 es hacer la foto del producto.

![](<../../.gitbook/assets/image (249).png>)

El tercer paso es hacer clic en el «check» de la parte inferior derecha de la pantalla para subir la foto.

{% hint style="info" %}
La cámara debe tener una calidad media, ya que hay un límite de 5 MB por foto.
{% endhint %}

![](<../../.gitbook/assets/image (256).png>)

Una vez cargada la foto, se reflejará en las pantallas web del Almacén P4.

![](<../../.gitbook/assets/image (122).png>)
