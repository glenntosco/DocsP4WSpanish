# Dimensiones del Producto

Las dimensiones del producto y los datos del código de barras pueden recogerse desde el PDT. Al entrar en la función PDT de dimensiones del producto y, a continuación, escanear el producto, se le presentarán las opciones para introducir los datos dimensionales del producto y los diferentes tamaños de envase (si el producto tiene tamaños de envase).

{% hint style="info" %}
Esta función suele delegarse en el Departamento de Control de Calidad.
{% endhint %}
