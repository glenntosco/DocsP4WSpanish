# Ubicación del producto

La opción Ubicación del Producto listará todas las ubicaciones/envases donde se puede encontrar el producto en el almacén, esto se utiliza típicamente para hacer una inspección visual de un producto específico.

{% hint style="info" %}
A menudo esta función está restringida por permisos para evitar que el usuario abuse del sistema.
{% endhint %}
