# Recepción de RMA

La recepción de RMA funciona exactamente igual que el proceso de recepción de PO, excepto que no se permite la sobre-recepción.

![P4 Warehouse Recepción de RMA](<../../.gitbook/assets/image (151).png>)

Inicie el proceso escaneando el código de barras del producto, después se le pedirá que introduzca la cantidad y, a continuación, la ubicación del contenedor.
