# Recepción sin RMA

La recepción sin RMA es el proceso que permite la recepción de productos que no están en el RMA.

{% hint style="info" %}
Usted puede recibir productos adicionales en un RMA existente, sin embargo no puede recibir productos para un cliente que no tiene RMA en el sistema.
{% endhint %}
