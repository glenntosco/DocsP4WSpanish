# Anular la recepción de RMA



En caso de que un producto se reciba de forma incorrecta y no se haya cerrado el RMA, existe la posibilidad de anular la recepción de los productos.

{% hint style="danger" %}
Recuerde que el proceso de recepción es absolutamente el proceso más importante de un almacén. Las malas prácticas de recepción provocarán muchos retrasos en otros procesos del almacén.
{% endhint %}
