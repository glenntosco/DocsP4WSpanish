# Lista RMA

Lista RMA mostrará una lista de pedidos RMA en espera de ser recibidos.

![P4 Warehouse Ordenes RMA ](<../../.gitbook/assets/image (251).png>)

Cuando el RMA llega al muelle, se debe presentar al receptor el número de RMA. El usuario de PDT seleccionará el RMA de la lista y procesará el RMA para asegurarse de que se ajusta al pedido previsto.
