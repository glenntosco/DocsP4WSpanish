# Recogida de paquetes completos

## Visión general de la recogida de paquetes completos

Para obtener información detallada sobre Full Pack Picking, póngase en contacto con su P4 Warehouse Partner.

{% hint style="warning" %}
Esto requiere etiquetas pre-impresas de LPN para completar este proceso. Todo el inventario de los pedidos debe estar en el almacén LPN al inicio del proceso.
{% endhint %}

La recogida de paquetes completos está diseñada para recoger cajas completas para envíos a granel.&#x20;

{% hint style="info" %}
En este estilo de picking, no se agita el pedido, simplemente se imprime el número de pedido en una etiqueta para iniciar el picking. Las etiquetas adecuadas se imprimirán según sea necesario. Este estilo puede ser «recogido» por varias personas a la vez.
{% endhint %}

Para procesar una orden «Full Pack Picking», el despachador web detendrá su proceso después del proceso de Asignación. Para este estilo de picking usted NO agita la orden. En la parte superior derecha de la pantalla hay un icono en forma de círculo donde encontrará una opción «Imprimir Código de Barras» aquí imprimirá tantas etiquetas como personas tenga procesando el pedido.

![P4 Warehouse Imprimir cualquier código de barras](<../../.gitbook/assets/image (207).png>)

Después de entregar las etiquetas de código de barras con el número de pedido al equipo de preparación de pedidos, éste introducirá en el PDT la opción de preparación de paquetes completos; en este punto, siga las instrucciones del PDT para procesar el etiquetado de los palés.
