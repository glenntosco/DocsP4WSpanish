# Recogida de cajas

## Recogida de cajas de cartón

Para obtener información detallada sobre el picking de cajas, póngase en contacto con su socio de P4 Warehouse.

Carton Picking está diseñado para el picking de cajas encartonadas, las etiquetas se asignan automáticamente a las cajas y las cajas se colocan en un LPN.

{% hint style="warning" %}
Se requieren etiquetas LNP preimpresas para este estilo
{% endhint %}

El Carton Picking o Cartonización es un estilo de picking en el que el Almacén P4 indica al preparador en qué tamaño de caja debe preparar el producto. El encartonado sigue un Perfil de Encartonado predefinido. Dependiendo de las regulaciones publicadas por su cliente Big Box, en cuanto a cómo se configurará el perfil de estuchado. Una vez que el perfil de estuchado es configurado y creado, es asignado al cliente en P4 Warehouse. Entonces cuando un despachador web está procesando pedidos para este cliente las reglas de estuchado serán seguidas para mantener su envío en cumplimiento con los requerimientos de sus clientes.

Cuando se imprimen las etiquetas de selección, el tamaño o el nombre de la caja se imprimirá en la parte superior derecha de las etiquetas de selección. El recolector tendrá que recuperar las cajas correctas del inventario de embalaje.
