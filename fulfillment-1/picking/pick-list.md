# Lista de recogida

La Lista de Recolección puede ser utilizada para permitir a los usuarios del PDT seleccionar la siguiente orden para Recolección, fuera de un centro de cumplimiento de eCommerce esta opción es normalmente deshabilitada ya que es responsabilidad del despachador Web decidir que órdenes son recolectadas.

![P4 Warehouse Lista de recogida](<../../.gitbook/assets/image (205).png>)

Desde la pantalla anterior, el usuario del PDT puede seleccionar cualquier orden para iniciar el proceso de picking.
