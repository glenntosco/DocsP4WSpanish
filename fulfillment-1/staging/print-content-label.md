# Imprimir etiqueta de contenido

Imprimir etiqueta de contenido sirve para imprimir el contenido de una caja de cartón o el contenido de un palé, dependiendo de cómo se haya tramitado el pedido. Durante el proceso de envío, el despachador puede elegir entre imprimir etiquetas de cartón o etiquetas de paleta, dependiendo de los procesos operativos y del tipo de pedido. Los envíos de Paquetes Pequeños normalmente se envían como pedidos de cartón, mientras que los pedidos de Carga de Camión y LTL normalmente se envían como palés.

{% hint style="info" %}
Si captura las dimensiones y el peso de la caja / paleta antes de imprimir la etiqueta de contenido, verá los datos de dimensiones y peso en la etiqueta de contenido.
{% endhint %}

![P4 Warehouse Contenido](<../../.gitbook/assets/image (188).png>)

Escanee la etiqueta de cartón / paleta para imprimir la etiqueta de contenido de cartón.
