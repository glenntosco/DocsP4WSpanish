# Reempaquetado



El reempaquetado es el proceso en el que:

* El producto se traslada de una caja a otra
* El producto a granel se separa en varias cajas
* El producto se traslada de una paleta a otra.

Reempaquetado de bolsas se puede encontrar en el menú de cumplimiento el menú de puesta en escena.

![](<../../.gitbook/assets/image (285).png>)

Inicie el proceso escaneando la etiqueta que tiene asignado el producto, a continuación podrá mover el producto.
