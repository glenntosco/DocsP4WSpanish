# Fusión de totes

Tote Merge permite al usuario combinar dos o más Cajas / Palets en una sola Caja / Palet. Se utiliza normalmente para combinar dos o más palés parciales en un solo pallet.

![P4 Warehouse Fusión de totes](<../../.gitbook/assets/image (135).png>)

Escanee la etiqueta «De la etiqueta Tote» y luego escanee la etiqueta «A la etiqueta Tote».
