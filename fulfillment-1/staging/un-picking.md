# Deshacer Picking

Puede ocurrir por varias razones, la más común es que los pallets no quepan en el remolque. En el caso de que los pallets no quepan, puede simplemente Des-recoger los pallets y colocar los productos en un contenedor.

{% hint style="danger" %}
Si se recogen varios pallets con una sola etiqueta de recogida, no existe ningún proceso para retirar un solo pallet. Es CRÍTICO que cada pallet tenga una etiqueta de pallet y que el producto se recoja correctamente según la etiqueta de pallet correspondiente.
{% endhint %}

![P4 Warehouse Cartón / Pallet](<../../.gitbook/assets/image (124).png>)

Escanee la etiqueta del contenedor (caja / pallet) que desea retirar.

![P4 Warehouse Escanear el producto no recogido a la papelera](<../../.gitbook/assets/image (221).png>)

Escanee el contenedor donde almacenará los productos hasta que se decida a qué contenedores se devolverán los productos.

![P4 Warehouse Pallet sin recoger](<../../.gitbook/assets/image (117).png>)

Como se puede ver en la imagen superior, el pallet se despaletiza y se devuelve a la ubicación A-44-A con una cantidad de 50.

{% hint style="info" %}
Los pedidos no se pueden retirar una vez cerrados. Asegúrese de sus procesos para poder retirar el producto antes de que se cierre el pedido.
{% endhint %}
