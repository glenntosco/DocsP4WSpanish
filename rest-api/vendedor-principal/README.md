# Vendedor principal

El siguiente segmento cubre las llamadas API que crean, actualizan/editan y eliminan un vendedor y recuperan una lista de todos los vendedores existentes.

Las llamadas a la API relativas al maestro de proveedores utilizan los «Id.» de proveedor (ejemplo: a79a50b4-4116-4525-8d35-43be41822250 ) generados por el sistema en el momento de la creación inicial del proveedor.

{% hint style="info" %}
Nota: El Vendor ID es único y creado automáticamente por el sistema. El Id. de proveedor no se puede modificar.
{% endhint %}
