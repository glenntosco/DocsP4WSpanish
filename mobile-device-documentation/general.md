# Información para Android Móvil

{% hint style="danger" %}
Es vital que su dispositivo Android tenga Google Play Store, P4 Warehouse requiere que la aplicación se actualice cuando se publique una nueva versión en Google Play Store. NO se distribuyen archivos APK.
{% endhint %}

La aplicación móvil es el caballo de batalla del sistema de almacén P4, dentro de la aplicación móvil es donde se ejecutan todas las instrucciones. La aplicación está diseñada para ser fácil de aprender y fácil de usar, lo que significa que los nuevos trabajadores de almacén pueden aprender el sistema rápidamente.

{% hint style="info" %}
La aplicación móvil del Almacén P4 funciona en Android Lollipop y superior, el software está diseñado para funcionar mejor en dispositivos Zebra Android. Esto no es un requisito firme, sin embargo, aprovechamos gran parte de la funcionalidad del software Zebra Data Wedge para controlar los dispositivos móviles. Data Wedge permite que el software requiera el escaneo de varios campos para evitar errores de tecleo.
{% endhint %}
