# Crear un producto

{% hint style="danger" %}
Configuración del producto
{% endhint %}

{% hint style="danger" %}
Aquí encontrará información más detallada sobre la configuración de productos. A continuación se muestra la configuración de un producto unitario simple sin atributos asignados al producto.
{% endhint %}

Para crear un producto unitario básico, abra Configuración > Productos

![P4 Warehouse Product List](<../.gitbook/assets/Product List.gif>)

En la pantalla de la lista de productos, puede seleccionar la(s) casilla(s) en la columna de la izquierda y luego hacer clic en imprimir. Esto imprimirá las etiquetas de los productos seleccionados.

Al hacer clic en el enlace de la columna SKU, se abrirá la pantalla de edición del producto.

Haciendo clic en el botón NUEVO puede crear un nuevo producto.

{% hint style="info" %}
Hacer números SKU o descripciones extremadamente largos no es beneficioso para su operación. Esto dará lugar a errores y a una ralentización de los procesos de trabajo. Además, los datos tendrán que caber en la etiqueta de código de barras, dependiendo del tamaño de la etiqueta y el texto que elija esto puede afectar a su operación.
{% endhint %}
