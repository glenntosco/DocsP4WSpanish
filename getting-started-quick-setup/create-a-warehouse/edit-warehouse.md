# Editar almacén

Aquí puede editar y añadir los datos necesarios para el almacén. Estos datos se reflejarán en las etiquetas de recogida y en los albaranes, por lo que es importante que rellene todos los datos posibles.

![Edit Warehouse](<../../.gitbook/assets/edit warehouse.gif>)

Cuando termine el proceso de añadir o editar los datos del almacén, pulse el botón Actualizar en la esquina superior derecha para guardar los cambios.

{% hint style="info" %}
Una vez que un almacén tiene transacciones, no puede borrarse sin borrar primero las ubicaciones y las zonas.
{% endhint %}
