# Crear un Almacén

**Configuración del Almacén**

{% hint style="info" %}
Recuerde que si P4Warehouse está integrado con un ERP, los códigos de almacén deben coincidir con el ERP. Consulte con su socio de P4 Warehouse antes de configurar los almacenes.
{% endhint %}

Configuración > Sistema > Almacén

![P4 Warehouse, Warehouse List](../../.gitbook/assets/warehouse.gif)

Pulse el botón para crear un nuevo almacén, pulse el enlace en clave para editar un almacén existente. Añada un nuevo almacén introduzca el código del almacén, el nombre del almacén y, si es necesario, la descripción.

![P4 Warehouse New Warehouse Setup](<../../.gitbook/assets/new warehouse.gif>)

Con el almacén creado, haga clic en el enlace de código de almacén para añadir información detallada del almacén.
