# Asignación de tareas

El Despachador Web puede asignar varias tareas a un usuario PDT, el usuario PDT oirá un sonido cuando se le asigne una nueva tarea. Se trata de una potente herramienta en el almacén, ya que se pueden asignar tareas a un usuario sobre la marcha, sin necesidad de localizar físicamente a la persona.
