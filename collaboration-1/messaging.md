# Mensajería - Chat

El chat del almacén es una potente herramienta utilizada en la comunicación con los usuarios de PDT para dar instrucciones en tiempo real. El chat es bidireccional, por lo que el usuario PDT también puede responder y solicitar instrucciones al Web Dispatcher.
