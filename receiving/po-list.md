# Lista de pedidos

PO List está diseñado para almacenes de tipo ecommerce con pocos empleados y fácil de identificar la información de las Órdenes de Compra del proveedor.

![Lista de pedidos Almacén P4](<../.gitbook/assets/PO List.png>)

* Después de seleccionar Recepción, seleccione Lista de pedidos.
* Aquí verá una lista de órdenes de compra abiertas, puede seleccionar la Orden de Compra para comenzar el proceso de recepción.
