# Código de barras del producto

El código de barras del producto permitirá al receptor imprimir etiquetas de código de barras para el producto al que le falten etiquetas antes de recibir el producto.

{% hint style="success" %}
Las etiquetas recibidas incluirán información clave en formato de código de barras, como el número de serie, el número de lote y la fecha de caducidad. A menudo, los productos no llevan los números de lote y las fechas en formato de código de barras del fabricante.
{% endhint %}

{% hint style="danger" %}
Es fundamental que todos los productos lleven una etiqueta con un código de barras antes de que finalice el proceso de recepción. Los productos sin código de barras serán muy difíciles (básicamente imposibles) de gestionar dentro del almacén, lo que provocará retrasos innecesarios en otros procesos como el picking.
{% endhint %}
