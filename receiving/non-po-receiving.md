# Recepción sin OP

La recepción sin pedido es para recibir artículos adicionales enviados por el proveedor, pero requiere un documento de compra básico.

{% hint style="info" %}
Esto es para recibir artículos adicionales en una Orden de Compra EXISTENTE. Esta función no es para recibir artículos sin una Orden de Compra.

Para recibir un artículo que no tiene una Orden de Compra para el proveedor tendrá que utilizar la función de ajuste.
{% endhint %}
