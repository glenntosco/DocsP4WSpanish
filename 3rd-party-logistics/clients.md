# Clientes 3PL

{% hint style="danger" %}
Si tiene previsto utilizar el Cliente multiempresa o 3PL, primero deberá configurar las pantallas 3PL.
{% endhint %}

En esta pantalla verá una lista de clientes 3PL o Multi Empresa, a continuación detallaremos el paso para determinar si la Empresa es 3PL o Multi Empresa.

![P4 Almacén 3PL Clientes / Multiempresa](<../.gitbook/assets/image (69).png>)

{% hint style="info" %}
En P4 Warehouse un cliente es un cliente 3PL o un Cliente Multiempresa, y Cliente se refiere a los clientes a los que usted envía producto.
{% endhint %}

En esta pantalla puede hacer clic en el botón Nuevo o en el nombre de la empresa para editar un cliente existente.

![P4 Warehouse Nueva configuración 3PL / Multiempresa](<../.gitbook/assets/image (269).png>)

En la pantalla sobre introduzca el nombre de la empresa, una descripción y los 7-9 dígitos SSCC Company id de la empresa. A continuación, haga clic en Enviar, esto creará el cliente 3PL / multiempresa. La pantalla le mostrará ahora una lista de clientes, haga clic en el nombre del cliente en la lista.

![Pantalla de configuración del cliente P4 Warehouse 3PL](<../.gitbook/assets/image (107).png>)

{% hint style="info" %}
La pantalla de abajo habilita la diferencia fundamental entre Cliente multiempresa y Cliente 3PL. Si se ha seleccionado la opción «Facturación activada», se convierte en un cliente 3PL.


{% endhint %}

![P4Almacén Habilitar cliente 3PL](<../.gitbook/assets/image (230).png>)

Facturación 3PL&#x20;

Una vez seleccionada la casilla Activar facturación el Cliente se convierte en un Cliente 3PL facturable. Dejando esta casilla desmarcada el cliente se comporta como multiempresa.&#x20;

Periodo Actual: le mostrará los cargos desde la última factura hasta la fecha actual.&#x20;

El Ciclo de Facturación es Mensual, Quincenal o Semanal.&#x20;

Inicio del Ciclo de Facturación: determina la fecha de inicio de la facturación, esto se establece una vez y se actualizará automáticamente a medida que se crean las facturas.&#x20;

Contabilización automática: Si se selecciona, la factura se contabilizará y se enviará el correo electrónico al cliente, al dejar desmarcada las facturas se crearán en borrador y aún en estado editable.
