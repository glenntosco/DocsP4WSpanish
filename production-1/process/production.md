# Producción

En este proceso sacará el producto acabado de la línea de producción para colocarlo en el inventario.

Para iniciar el proceso de retirada de producción, escanee el número de orden de trabajo en la pantalla que aparece a continuación.

![P4 Warehouse Tirada de producción](<../../.gitbook/assets/image (88).png>)

En la siguiente pantalla se le pedirá la cantidad de productos terminados que está retirando de la línea de producción y trasladando al almacén.

![P4 Warehouse Cantidad de tirada de producción](<../../.gitbook/assets/image (202).png>)

Introduzca sólo la cantidad que va a trasladar físicamente al almacén.

{% hint style="info" %}
Si está trasladando un palé que contiene veinticinco cajas, introduzca sólo veinticinco en la pantalla anterior.
{% endhint %}

Después de introducir veinticinco en la pantalla anterior se le preguntará qué ubicación utilizará para almacenar el producto terminado, escanee la ubicación de la ubicación y luego regrese al área de producción por la segunda paleta para completar el ejemplo anterior.

![P4 Warehouse Localizar el producto producido](<../../.gitbook/assets/image (153).png>)
