# Lista de órdenes de trabajo

La Lista de órdenes de trabajo muestra una lista de las órdenes de fabricación que están listas para ser producidas.

![P4 Warehouse Lista de órdenes de trabajo](<../../.gitbook/assets/image (165).png>)

En la pantalla puede seleccionar la Orden de Trabajo para iniciar el proceso de producción.

![P4 Warehouse Recogida de materias primas](<../../.gitbook/assets/image (275).png>)

En esta pantalla comenzará el proceso de recogida de la mercancía en bruto y el traslado de esta mercancía a la zona de producción asignada.
