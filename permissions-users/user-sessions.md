# Sesiones de usuario

\
La sección Sesiones de Usuario proporciona una lista sencilla de información sobre la actividad básica de los distintos usuarios dentro del Sistema de Gestión de Almacenes (SGA). El nombre de la cuenta de usuario, Nombre, Cliente y Rol aparecerán todos completados dentro del perfil de ese usuario. La IP cambiará en función del dispositivo o la red desde la que el usuario haya accedido al sitio web del SGA. La columna Creado detalla la duración de la sesión del usuario. La columna Última actividad detalla el tiempo transcurrido desde la última vez que el usuario inició sesión en el WMS. Utilice la columna Acción para cerrar la sesión de un usuario correspondiente del sistema.
