# Generador de consultas

Consulta la base de datos para ver datos o crear informes escribiendo TSQL y haciendo clic en ejecutar.

![P4 Warehouse Generador de consultas](<../.gitbook/assets/system querry.jpg>)

Los resultados de la consulta pueden exportarse a formato Excel.

{% hint style="info" %}
En esta pantalla el P4 Warehouse sólo aceptará comandos de «seleccionar», no podrá actualizar o borrar datos desde esta pantalla.
{% endhint %}
