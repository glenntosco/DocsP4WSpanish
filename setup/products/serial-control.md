# Control por serie

Productos controlados en serie

Después de crear el producto básico como se indica aquí. Seleccionará el atributo serie controlada. Si su producto siempre llega con la misma secuencia para el número de serie, puede añadir una máscara de número de serie para evitar escanear accidentalmente el código de barras equivocado en el campo de número de serie. En la foto de abajo la máscara es DJ más cinco dígitos. Los dígitos desconocidos se indican con un guión bajo \_.

Ejemplo: DJ\_\_\_\_\_



![Configuración del número de serie del almacén P4](<../../.gitbook/assets/image (84).png>)

La configuración de los atributos debe seleccionarse antes de realizar cualquier transacción para este producto en el sistema.
