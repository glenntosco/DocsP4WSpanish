# Control decimal

Los productos controlados decimalmente incluyen los siguientes tipos de productos

* Kilogramos
* Gramos
* Libras
* Onzas
* Litro
* Mililitro
* Metro
* Centímetro
* Milímetro
* Pies
* Pulgada

Tenga en cuenta que debe ajustarse a la unidad que utilice para vender el producto.

{% hint style="danger" %}
La configuración de los atributos debe seleccionarse antes de realizar cualquier transacción para este producto en el sistema.
{% endhint %}
