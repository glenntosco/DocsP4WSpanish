# Lista de materiales

La lista de materiales en P4 Almacén es para el área de producción dentro del WMS.

Después de crear un nuevo Producto seleccione el atributo «Lista de Materiales» Verá una nueva opción debajo del área de atributos para añadir los subcomponentes a la Lista de Materiales.





![P4 P4 Lista de materiales de almacén para la producción](<../../.gitbook/assets/image (322).png>)

Después de añadir los subcomponentes, haga clic en el botón de actualización de la esquina superior derecha para guardar los cambios.

{% hint style="warning" %}
La producción es el proceso de transformación de materias primas en una nueva SKU.
{% endhint %}

{% hint style="success" %}
P4 Warehouse incluye Product Bundle para grupos de productos que no son de producción.

&#x20;Ejemplo: Un juego de comedor que incluye una base de mesa, un tablero y cuatro sillas es un Paquete de productos.&#x20;

Ejemplo: La fabricación de pan a partir de materias primas es producción.
{% endhint %}
