# Control de lotes

El control de lotes es un componente vital en la trazabilidad de los productos en la cadena de suministro. En la pantalla de edición del producto seleccione la casilla Lote Controlado, si es necesario puede introducir el Patrón de Lote. El patrón de LOTE ayudará a prevenir errores en la entrada de datos de LOTES.

Ejemplo de patrón de LOTE: Su número de LOTE para este artículo es BB-21xxxx, siendo la x el mes y el día del LOTE de producción.

El patrón del LOTE sería BB-21\_\_\_\_ (4 guiones bajos) con el patrón en su lugar el lote introducido debe seguir siempre esta regla.

{% hint style="danger" %}
La configuración de los atributos debe seleccionarse antes de realizar cualquier transacción para este producto en el sistema.
{% endhint %}
