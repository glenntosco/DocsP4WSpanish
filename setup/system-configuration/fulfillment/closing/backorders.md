# Pedidos pendientes

Esta pantalla de configuración permite configurar los pedidos pendientes.

1. Generar pedido pendiente al cierre, si se selecciona esta opción se creará automáticamente un nuevo pedido para cualquier producto que no se haya enviado en el pedido original.
2. Auto Liberar pedido pendiente, si se selecciona, moverá el nuevo pedido pendiente de borrador a estado listo para asignar.
