# Conteo de ciclos

Autorización de recuento cíclico

En Setup > System > Configuration > Business esta casilla de verificación decidirá si el Almacén P4 aplica automáticamente los Conteos de Ciclo o si el Conteo de Ciclo debe ser aprobado primero.

{% hint style="warning" %}
Si marca esta casilla y hace clic en Guardar, el recuento de ciclos no se aplicará o, en el caso de una interfaz ERP, el recuento de ciclos no se aplicará ni se cargará hasta que lo apruebe un despachador web.
{% endhint %}

{% hint style="success" %}
Si deja esta casilla sin marcar, se aplicarán los recuentos de ciclo y, si procede, se cargarán en su ERP.
{% endhint %}
