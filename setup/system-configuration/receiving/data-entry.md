# Entrada de datos

Esta pantalla permite la configuración para la recepción.

* Prefijo del número de pedido: es el prefijo de los pedidos de compra, PO- es un prefijo estándar.
* Comportamiento de las sku duplicadas
  * Permitir: puede tener el mismo SKU en la orden de compra más de una vez.
  * Aviso: avisa al usuario de que la sku ya está en el pedido, pero le permite añadirla varias veces.
  * Impedir: no permite al usuario añadir un sku duplicado al pedido.

{% hint style="success" %}
Prevenir es la configuración más habitual.
{% endhint %}
