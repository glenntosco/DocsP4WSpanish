# Impresión de documentos

Esta pantalla le permitirá establecer un descargo de responsabilidad estándar para el Documento de Recepción, esta es una configuración muy importante especialmente para los proveedores 3PL.

![P4 Warehouse Descargo de responsabilidad](<../../../.gitbook/assets/image (233).png>)

{% hint style="info" %}
Después de introducir el texto de la cláusula de exención de responsabilidad, haga clic en Guardar.
{% endhint %}
