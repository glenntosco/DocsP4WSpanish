# Dispositivos portátiles

En esta pantalla verá la configuración que afecta directamente a lo que el usuario de la Handheld ve durante el proceso de recepción.

![](<../../../.gitbook/assets/image (137).png>)

Opciones:

* Mostrar la cantidad esperada: Esto es a veces también llamado Recepción Ciega, dependiendo de las políticas de su Compañía como esta opción es configurada. En un almacén típico a menudo es mejor forzar al receptor a contar los productos que realmente han llegado en lugar de que el receptor simplemente acepte la cantidad esperada.
* Permitir que el usuario cierre: Esta configuración muestra un botón de cierre para el usuario de mano, permitiéndole cerrar una Orden de Compra cuando todo el producto para la Orden de Compra ha sido recibido.
  * Esta opción está desactivada por defecto, ya que puede provocar errores irreversibles si el pedido se cierra antes de recibir el producto.
* Peso y dimensiones de la recogida: Este es un ajuste vital si el peso y las dimensiones del producto son importantes para su operación. Si se activa en la recepción, si el producto carece de peso y dimensiones, el dispositivo de mano pedirá al receptor que introduzca los datos correctos.
  * Esta información no se solicitará si el maestro de productos ya está cumplimentado.
* Recoger código de barras: Este ajuste es para recoger los datos correctos del código de barras, este ajuste pedirá al receptor que escanee los códigos de barras para el producto / packsizes si los datos del código de barras faltan en el sistema.
  * Esta información no se solicitará si el maestro de productos ya está cumplimentado.
* Imprimir Etiquetas, esta configuración preguntará al receptor si desea imprimir etiquetas después de recibir una línea de orden de compra. Esta es una herramienta muy poderosa cuando tiene productos que llegan sin etiquetas de código de barras.
  * El mando preguntará primero al receptor si necesita imprimir etiquetas, cuando el receptor responda afirmativamente, el sistema le preguntará cuántas etiquetas debe imprimir.
* Mostrar ubicaciones de productos, este ajuste mostrará al receptor en la computadora de mano dónde está almacenado actualmente el producto en su almacén, o dónde se almacenó el producto por última vez en su almacén. De lo contrario, el receptor localizará el producto en una ubicación de ubicación que esté vacía.
