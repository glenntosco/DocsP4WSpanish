# Notificación por correo electrónico

Esta pantalla le permitirá enviar automáticamente un pdf a los destinatarios de su elección cuando se cierre una Orden de Compra.

{% hint style="info" %}
Manteniendo el ratón sobre los campos, verá las variables que están disponibles para su uso.
{% endhint %}

![P4 Warehouse Variables de correo electrónico](<../../../.gitbook/assets/image (247).png>)

{% hint style="info" %}
Un ejemplo de variable en el campo Para: son @CorreoVendedor o @CorreoCliente o simplemente introduzca la dirección de correo electrónico de su departamento de recepción.
{% endhint %}

![P4 Warehouse Configuración de la notificación de recepción](<../../../.gitbook/assets/image (80).png>)
