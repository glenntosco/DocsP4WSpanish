# Códigos de motivos

Esta página de configuración incluye Ajuste de entrada, Ajuste de salida, Retirada, Devolución de cliente y Devolución de proveedor. Dentro de cada campo se puede introducir una lista separada por comas de códigos de motivo para cada campo. Tras modificar cualquier entrada, seleccione el botón «Guardar» situado en la parte inferior derecha.

{% hint style="info" %}
Estos códigos de motivo se mostrarán en el dispositivo móvil al ajustar las cantidades de producto.
{% endhint %}
