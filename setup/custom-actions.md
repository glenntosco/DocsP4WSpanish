# Acciones personalizadas

Las Acciones Personalizadas son un método para añadir funciones al sistema de gestión de almacenes, las Acciones Personalizadas se utilizan a menudo en almacenes 3PL para cobrar a un cliente por cosas como gastos de envío o envoltura de palets. Sin embargo, esta es una poderosa función que puede ser utilizada para generar un registro de cualquier actividad extra realizada en cualquier tipo de documento. Las Acciones Personalizadas también pueden activar la impresión de tipos específicos de etiquetas como, por ejemplo, etiquetas de exportación.
