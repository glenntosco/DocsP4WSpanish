# Contenedores

Las ubicaciones son el motor de un sistema de gestión de almacenes. Las ubicaciones permiten crear un pequeño espacio en el que se almacena el producto para que los usuarios encuentren fácilmente el producto correcto en un periodo de tiempo muy eficiente.

Sistema>Almacenes>Almacenes>Contenedor

{% hint style="info" %}
No puede añadir una zona sin haber seleccionado antes el almacén.
{% endhint %}

![P4 Warehouse Configuración de zona](<../../.gitbook/assets/image (194).png>)

En la pantalla anterior, seleccione la Zona que desea editar o añadir.

![P4 Warehouse Configuración de zonas](<../../.gitbook/assets/image (59).png>)

Las ubicaciones de contenedores suelen heredar los atributos del nivel de zona. Las ubicaciones pueden configurarse una a una en caso de que una ubicación específica tenga una configuración diferente a las demás ubicaciones de la zona.
